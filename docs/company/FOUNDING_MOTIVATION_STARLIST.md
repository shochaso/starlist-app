# 創業の動機（STARLIST）

私はこれまで事業のアイデア自体は数多く持っていましたが、自分一人で具体的なサービスとして形にする手段がなく、創業には至っていませんでした。近年、生成AIをはじめとしたAI技術の発展により、個人でも高度なサービスを設計・試作できる環境が整い、「自分の頭の中の構想を、自分の手で形にできる」と実感できるようになったことが、今回の創業を決意した大きなきっかけです。

STARLIST（スターリスト）を立ち上げようと考えた背景には、私自身が以前から抱いていた「市場への違和感」があります。一般的には「お金は、人にどれだけ幸福をもたらしたかの対価だ」と言われますが、現実はそうなっていない場面を多く見てきました。学生時代や職場でも、容姿端麗で、その場に「いるだけ」で周囲を明るくし、幸せな気持ちにさせる人たちがいました。しかし、その「存在そのもの」がもたらす価値については、報酬が一切支払われていません。まるで、豊富な湯水が湧き続けているのに、水力発電や温泉施設などに活かされることもなく、ただ流れ続けているような状態だと感じていました。本来であれば、そのリソースを提供している本人にも、適切なリターンが還元されるべきだと考えています。

一方で、容姿やキャラクターといった「人としての魅力」を活かして収益化しようとすると、現状の選択肢として思い浮かぶのは、芸能人・アイドル・モデル・YouTuber・インフルエンサー・ライバー・コンパニオン・水商売などであり、いずれも「顔や体をさらすこと」が前提になりがちです。動画撮影・編集、投稿の企画づくりといった時間的・学習的コストも大きく、さらに炎上・誹謗中傷・身バレやストーカー被害といった社会的リスクも無視できません。その結果、すでにSNSで一定のフォロワーを持ち、影響力があるにもかかわらず、「本業との両立」「身バレ」「精神的負荷」などを理由に、一歩踏み出せない人が多い構造になっていると感じていました。

STARLISTでは、いわゆる芸能プロ所属タレントに限らず、SNS等で一定のフォロワーを持つ学生・会社員・主婦などのマイクロインフルエンサー層を主な対象としています。すでに一定の発信力やファンを持ちながらも、「本格的な活動に踏み切るほどの時間も勇気もない」「収益化の手段が広告や案件に限られている」といった人たちに対し、より低リスク・低負荷で始められる第2の収益軸を提供したいと考えました。

他方で、私自身は熱心な芸能人オタクというわけではありませんが、学生時代を振り返ると、好きな人や「かわいいと思うクラスメイト」がいると、どんなアーティストが好きなのか、どんな食べ物が好きなのかといった「ささやかな日常」を強く知りたいと思っていました。廊下ですれ違うだけの高嶺の花のような存在とは、そもそも話すきっかけがなく、好みを知る術もありませんでした。それでも、「本来なら知り得なかったはずのその人の日常の一部を、対価を払ってでも知りたい」と感じる人は、自分以外にも必ずいると考えました。こうした「すでに認知している相手の、よりプライベートに近い日常へアクセスしたい」というファン心理こそが、STARLISTの根本的な発想につながっています。

こうした問題意識から、私は「従来型のマネタイズ方法」と「STARLISTのモデル」の違いを明確に意識するようになりました。従来の方法では、動画撮影・写真撮影・店舗への出勤・SNS投稿・文章作成など、新たな労働が必須であり、

- 顔出しリスクが高い  
- 労働負荷が大きい  
- 時間的制約が厳しく、休むと収益が止まる  
- 専門知識やスキルが必要  
- 炎上・誹謗中傷リスク、加齢による不利など、社会的リスクも大きい  

といった課題があります。

これに対してSTARLISTは、「日常生活で生まれている情報」を公開するだけで収益化できる仕組みを目指しています。具体的には、観たYouTube動画、聴いている音楽、コンビニやECで買った商品、普段使っているコスメや洋服など、すでに行っている消費行動や視聴行動を、そのまま「生活ログ」として投稿するだけで、ファンが月額課金で閲覧できるモデルです。

- 顔出し不要・ハンドルネーム前提で運用可能  
- 自宅でも外出先でも、空き時間に簡単に投稿できる  
- 家で寝ながら動画を見ている時間さえ、将来的な収益のタネになる  
- 専門的なスキルが不要  
- コメント投稿や閲覧を有料会員に限定したクローズドな仕組みにすることで、炎上・誹謗中傷・身バレリスクを抑えられる  
- 画像や動画で自身を「見せること」が前提ではないため、加齢を過度に気にせず継続的にマネタイズできる  

このように、STARLISTは「コンテンツを新しく作る仕事」ではなく、「すでに行っている生活そのものを収益化する仕組み」であり、従来にはなかった“生活収益化プラットフォーム”だと考えています。特に、SNS等で一定のフォロワーを持つマイクロインフルエンサー層にとっては、既存フォロワーとの関係性を活かしながら、無理のない範囲で第2・第3の収益源を持てる仕組みになると考えています。

私はこれまで、一般のファンとしてYouTubeやSNSを日常的に利用しつつ、「日常をもっと知りたい」という感情と、「その価値を提供する人に、正当なリターンが返るべきだ」という思いを強く持ってきました。また、合理主義的な価値観やお金・時間の使い方への関心から、AIやGPTを活用してビジネスを構築することにも強い問題意識とモチベーションを持っています。「埋もれた存在価値に経済的な意味を与えたい」という市場への違和感と、「フォロワーを持つ個人と、そのファン心理を自分自身が体験してきたこと」、そして「AIの発展により、それを現実のサービスとして実装できる環境が整ったこと」が重なった今こそ、自分がSTARLISTを立ち上げる必然性があると考え、創業を決意いたしました。
