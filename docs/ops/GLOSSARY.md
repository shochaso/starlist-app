# Glossary（用語集）

## Go/No-Go

最終判定ゲート。10項目の統合チェックロジックで判定。

## Artifacts

CIで保存する原本（Git非管理）。JSON/ログ等の機微情報を含む。

## Front-Matter

監査票ヘッダ（機械読取）。JSON Schemaで検証可能。

## Safe Mode

送信停止・監査のみ実行モード。`STARLIST_SEND_DISABLED=1`で有効化。

## Permalink

Slack投稿への固定リンク。監査票とSlack投稿を結ぶ証跡。

## p95 latency

レイテンシの95%点。SLO目標値は2000ms以内。

## SLO

Service Level Objective。サービスレベル目標。

## RACI

責任分担表。Accountable/Responsible/Consulted/Informedの4役割。
