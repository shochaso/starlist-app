# データ保持/削除ポリシー（監査/機微）

## Artifact保持

* **最低保持期間**: 120日
* **延長**: 必要に応じて延長可能
* **保存先**: GitHub Actions Artifacts（Git非管理）

## Git管理

* **監査票（MD）**: Git管理対象
* **原本JSON/ログ**: Git非管理（Artifactsのみ）

## 機微情報

* **常時レダクション**: メール・電話・カード番号等を自動置換
* **Stripe顧客識別子**: 要約のみ保存

## ダッシュボードKPI（集計値のみ）

* **対象外**: `dashboard/data/latest.json` は集計値のみで、PIIを一切保持しない
* **保持期間**: Git管理対象（長期保持）
* **更新頻度**: CI成功時に自動更新
