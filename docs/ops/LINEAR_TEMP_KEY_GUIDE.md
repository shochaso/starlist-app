# Linear APIキー一時使用手順（安全版）

## 📋 ステップ1: 一時キーを作成

1. Linearにログイン: https://linear.app/settings/security
2. 「Personal API keys」→「Create new key」
3. キー名: 「一時 - STA-8作成用」など
4. 生成されたキーをコピー

## 📋 ステップ2: キーを共有

- キーをここに貼り付けてください
- こちらでSTA-8を作成します

## 📋 ステップ3: STA-8作成完了後、キーを削除

1. Linearの設定ページに戻る: https://linear.app/settings/security
2. 「Personal API keys」セクションを開く
3. 作成した一時キーの横の「削除」ボタンをクリック
4. 確認して削除

これで、キーは完全に無効化され、バックアップに残っていても使用できません。

---

## 🔒 なぜ安全か

- ✅ キーは短時間のみ有効
- ✅ 使用後すぐに削除できる
- ✅ 削除後は完全に無効化される
- ✅ バックアップに残っていても使用不可
