# アーキテクチャ図ファイル

このディレクトリには、StarlistプロジェクトのER図・シーケンス図・アーキテクチャ図を配置します。

## 命名規則

`[type]_[name]_[version].{mermaid|png}`

- **type**: `er`（ER図）、`seq`（シーケンス図）、`arch`（アーキテクチャ図）
- **name**: 図の名前（スネークケース）
- **version**: バージョン番号（`v1`, `v2`等）

## 形式

- **Mermaid**（推奨）: `.mermaid`拡張子、テキスト形式で編集可能
- **PNG**（静的画像）: `.png`拡張子、Mermaidから生成可能

## 更新手順

1. Mermaidファイルを編集
2. プレビューで確認（VS CodeのMermaid拡張、または`mermaid-cli`でPNG生成）
3. `docs/overview/COMMON_DOCS_INDEX.md`の図ファイル一覧を更新
4. 変更をコミット

## 参考

- `docs/overview/COMMON_DOCS_INDEX.md` - 図ファイル一覧とリンク
- `docs/Mermaid.md` - ドキュメント相関図（Mermaid形式）
